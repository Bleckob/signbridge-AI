## 3. WebSocket Connection Guide

### Who Uses This?
- **Confidence** — sends audio chunks to the server
- **David** — receives animation data from the server

### How to Connect
WebSocket URLs start with `ws://` NOT `http://`.  
You cannot open them in a browser — use them in your code only.
```
ws://127.0.0.1:8000/ws/{session_id}
```

Replace `{session_id}` with the unique session ID generated by David's frontend.

### How Session ID Works
1. David generates a unique session ID using `crypto.randomUUID()`
2. David passes it to Confidence
3. Both use the same session ID when connecting

### Example Connection (JavaScript)
```javascript
const sessionId = crypto.randomUUID()
const socket = new WebSocket(`ws://127.0.0.1:8000/ws/${sessionId}`)

socket.onopen = () => {
    console.log("Connected to SignBridge AI ✅")
}

socket.onmessage = (event) => {
    const data = JSON.parse(event.data)
    console.log("Message received:", data)
}
```

### Welcome Message
When you connect successfully you will receive:
```json
{
    "type": "connection_established",
    "session_id": "your-session-id",
    "message": "Connected to SignBridge AI ✅"
}
```

## 4. Redis Stream Names

These are the conveyor belts that carry data between services.  
Use these EXACT names — spelling mistakes will break the pipeline.

| Stream Name | Carries | From | To |
|------------|---------|------|----|
| `audio-chunks` | Raw audio data | Confidence | Ife |
| `asr-output` | Converted text | Ife | Amos |
| `nlp-output` | Sign language gloss | Amos | Blessing |
| `session-result` | Animation data | Blessing | David |

### How to Push to a Stream (Python)
```python
import redis

r = redis.Redis(
    host=REDIS_HOST,
    port=REDIS_PORT,
    username=REDIS_USERNAME,
    password=REDIS_PASSWORD,
    decode_responses=True,
    ssl=False
)

# Push data onto a stream
r.xadd("audio-chunks", {
    "session_id": "your-session-id",
    "data": "your-data-here"
})
```

### How to Read From a Stream (Python)
```python
# Read messages from a stream
messages = r.xread(
    {"asr-output": "0"},
    count=10,
    block=0
)
```

## 6. Data Flow
```
User speaks into mic
        ↓
Confidence captures audio
        ↓
Confidence pushes to Redis stream: audio-chunks
        ↓
Ife reads from: audio-chunks
Ife converts speech to text
Ife pushes to Redis stream: asr-output
        ↓
Amos reads from: asr-output
Amos converts text to sign gloss
Amos pushes to Redis stream: nlp-output
        ↓
Blessing reads from: nlp-output
Blessing looks up pose data from Supabase
Blessing pushes animation to Redis stream: session-result
        ↓
David reads animation from: session-result
David's avatar signs to the user 🤟
```

## 8. Common Issues and Fixes

| Problem | Fix |
|---------|-----|
| Server not responding | Run `taskkill //F //IM python.exe` then restart |
| Redis disconnected | Check your `.env` values are correct |
| WebSocket not connecting | Make sure URL starts with `ws://` not `http://` |
| Module not found error | Make sure your venv is activated |
